## Design Principles

### Visual Style
- **DaisyUI Components:** Cards, buttons, badges, hero, stats, testimonials, footer
- **Theme:** Black and white color scheme (light theme: white/light gray, dark theme: black/dark gray)
- **Logo:** Jarga logo top right on page (jarga-logo-light-trans.png for light theme, jarga-logo-dark-trans.png for dark theme)
- **Typography:** Clear hierarchy, readable fonts, generous spacing
- **Icons:** Heroicons or similar for consistency
- **Spacing:** Generous whitespace, breathable sections

### Responsive Design
- Mobile-first approach
- Test on tablet and desktop
- Touch-friendly CTAs
- Readable on all screen sizes

### Performance
- Fast page load
- Optimized images
- Minimal JavaScript
- Progressive enhancement

### Accessibility
- Semantic HTML
- ARIA labels where needed
- Keyboard navigation
- Color contrast compliance
